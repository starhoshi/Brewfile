Brewfile
========

Package list for Homebrew

## import

```sh
$ git clone git@github.com:starhoshi/Brewfile.git
$ brew tap Homebrew/bundle
$ brew bundle --file=Brewfile/Brewfile
```

## export

```sh
$ brew bundle dump --force
```
